How-To Host a Facebook Contest

This guide outlines how to use Facebook's Graph API to host a
promotional contest, without having to manually track the users who
liked your post.

Overview

Part 1: Setting Up and Using Facebook's Graph API

In order to run a fully automated Facebook contest, you will need to be able to obtain an Access Token, make HTTP GET requests and utilize Page and Post ID's. This section covers each of those things!

Go to Part 1

Part 2: Tracking Post Likes

When you post your contest, you will need to be able to track the number of users who like it. This section explains the process of tracking post likes so that we can determine the number of participants in our contest.

Go to Part 2

Part 3: Identifying Participants

Along with the number of participants, we will also need the user name and ID of each participant so that we can name a winner at the end. This section outlines how to display a list of participants including their user ID.

Go to Part 3

Part 4: Determining Eligibility

The entire purpose of hosting a contest is to acquire page likes, therefore we want to be able to assure that a participant is only eligible to win the contest if the like both your contest post AND your page.

Go to Part 4

Part 5: Declaring A Winner

Now that we have a list of eligible participants, all we need to do is randomly choose a winner. This section explains just how to do that!

Go to Part 5
